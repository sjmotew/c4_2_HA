# Should you leave Control4?

Before you spend a dollar or pull a single zone, get honest about whether this migration is right for *you*. Control4 is a competent system, and leaving it means trading a paid integrator for owning the platform yourself. That's a real trade with real upside and real cost.

- [**What you're signing up for**](what-youre-signing-up-for.md) — The realistic time, skills, and cost of the move, plus the family-acceptance factor.
- [**Is this for you?**](is-this-for-you.md) — A short readiness self-assessment.
